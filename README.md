  <h2 id="about">About</h2> 
  <p>This project provides a variety of recipes specifically designed for people with diabetes. It aims to offer delicious and healthy meal options that help manage blood sugar levels.</p> 
  
  <h2 id="features">Features</h2> 
  <ul> 
    <li>200 diabetes-friendly recipes</li> 
    <li>Easy-to-follow instructions</li> 
    <li>Nutrition information for each recipe</li> 
    <li>Categories for different meal types (breakfast, lunch, dinner, snacks)</li> 
  </ul> <h2 id="installation">Installation</h2> 
  <pre><code> git clone https://github.com/Eduardo-J-Morales/200-Diabetes-Recipes.git cd 200-Diabetes-Recipes npm install </code></pre> 
  <h2 id="usage">Usage</h2> <pre><code> npm start </code></pre> 
  
  <p>Open your browser and navigate to <a href="http://localhost:3000">http://localhost:3000</a> to explore the recipes.</p>
